> 难度等级: 低  
> 预计耗时: 5-10分钟

---

## 案例 F1: 浏览器检索热点新闻信息

### 场景描述
使用agent-browser skill自动化检索热点新闻信息，验证OpenClaw的浏览器自动化能力。

### 前置要求
- 已安装 `agent-browser` skill
- 系统可访问互联网

### 测试提示词

```
【测试案例 F1】浏览器检索热点新闻信息

请按照以下步骤完成任务：

Step 1 - 环境检查
1. 确认agent-browser skill已安装
2. 如未安装，请执行: 请帮我安装agent-browser skill

Step 2 - 执行任务
请使用浏览器完成以下操作：
1. 打开浏览器访问 "https://www.baidu.com"
2. 搜索 "今日A股热点新闻"
3. 打开前3条搜索结果
4. 提取每条结果的标题和摘要
5. 总结今日A股市场的3个主要热点话题

Step 3 - 输出结果
1. 将检索结果整理为结构化文本
2. 保存至: /workspace/openclaw_research_test/outputs/data/F1_news_summary.md
3. 返回热点话题摘要

【预期输出】
- 3条热点新闻标题和摘要
- 结构化Markdown文件
- 执行过程日志
```

## 案例 F2: 查询并安装热门skills

### 场景描述
测试OpenClaw的skill发现和安装能力，从ClawHub查询并安装热门skill。

### 前置要求
- 可访问ClawHub (https://clawhub.ai)
- 网络连接正常

### 测试提示词

```
【测试案例 F2】查询并安装热门skills

请按照以下步骤完成任务：

Step 1 - 查询热门skills
1. 访问 https://clawhub.ai
2. 查看当前热门的skills列表
3. 找出与"股票分析"相关的3个热门skill

Step 2 - 安装测试
选择其中一个skill进行安装测试：
- 优先选择: stock-watcher 或 stock-analysis
- 安装命令: "请帮我安装并启用这个skill: [skill链接]"

Step 3 - 功能验证
1. 确认skill安装成功
2. 查看skill的功能说明
3. 执行一个简单的功能测试

Step 4 - 输出结果
1. 记录查询到的热门skills列表
2. 记录安装过程和结果
3. 保存至: /workspace/openclaw_research_test/outputs/data/F2_skill_installation.md

【预期输出】
- 热门skill列表（至少3个）
- 安装成功确认
- skill功能说明
```

## 案例 F4: 本地文件管理与信息检索

### 场景描述
测试OpenClaw的本地文件操作能力，包括文件创建、搜索、读取和整理。

### 前置要求
- 文件系统访问权限
- 测试目录可写

### 测试提示词

```
【测试案例 F4】本地文件管理与信息检索

请按照以下步骤完成任务：

Step 1 - 创建测试目录结构
在 /workspace/openclaw_research_test/temp/test_f4/ 下创建以下目录结构：
```
reports/
  ├── 2026/
  │   ├── Q1/
  │   └── Q2/
  └── 2025/
      ├── Q3/
      └── Q4/
data/
  ├── raw/
  └── processed/
```

Step 2 - 创建测试文件
1. 在 reports/2026/Q1/ 下创建文件 "market_review_jan.md"
   内容: "# 2026年1月市场回顾\n\n## 主要指数表现\n- 上证指数: +3.2%\n- 深证成指: +2.8%\n"

2. 在 data/raw/ 下创建文件 "stock_list.csv"
   内容: "代码,名称,行业\n600519,贵州茅台,食品饮料\n000858,五粮液,食品饮料\n"

Step 3 - 文件搜索测试
1. 搜索所有.md文件
2. 搜索包含"市场"关键词的文件
3. 列出reports目录下的所有文件

Step 4 - 文件读取与整理
1. 读取market_review_jan.md内容
2. 读取stock_list.csv内容
3. 将两个文件内容整合为一个摘要

Step 5 - 输出结果
1. 将操作日志保存至: /workspace/openclaw_research_test/outputs/data/F4_file_operations.log
2. 将整合摘要保存至: /workspace/openclaw_research_test/outputs/data/F4_summary.md

【预期输出】
- 完整的目录结构
- 测试文件内容
- 搜索结果
- 整合摘要
```

## 案例 F5: 自动化任务提醒

### 场景描述
测试OpenClaw的定时任务和提醒功能，设置自动化任务提醒。

### 前置要求
- 已安装 `cron` skill
- 通信渠道已配置（飞书/QQ/Telegram等）

### 测试提示词

```
【测试案例 F5】自动化任务提醒

请按照以下步骤完成任务：

Step 1 - 检查cron skill
1. 确认cron skill已安装
2. 如未安装，请先安装

Step 2 - 创建定时提醒任务
创建一个测试用的定时提醒任务：
- 任务名称: "测试提醒"
- 提醒内容: "这是一个OpenClaw自动化任务提醒测试"
- 执行时间: 5分钟后（用于快速测试）

Step 3 - 验证任务设置
1. 列出所有定时任务
2. 确认任务已正确设置
3. 等待任务执行或手动触发测试

Step 4 - 输出结果
1. 记录任务设置过程
2. 记录任务执行结果
3. 保存至: /workspace/openclaw_research_test/outputs/data/F5_cron_task.md

【预期输出】
- 定时任务列表
- 任务执行确认
- 提醒消息接收确认

【注意事项】
- 实际提醒需要通信渠道已正确配置
- 如未配置通信渠道，仅验证任务设置功能即可
```
