# F11 测试规范

## 测试名称

F11 同花顺 API 接口配置与公告数据提取

## 测试目标

验证 OpenClaw 能否在 Linux 环境下：

1. 使用同花顺官方 SDK 完成安装
2. 使用用户提供的账号密码完成登录配置
3. 测试接口可用性
4. 提取平安银行自 2026-01-01 以来的收盘价数据
5. 提取中国神华自 2026-02-13 以来的公告日期、公告标题、公告链接

## 安全要求

1. 账号密码只能作为运行期配置使用，不能写入仓库文件。
2. 报告中不得回显完整账号密码。
3. 如需记录配置状态，只记录“已配置/未配置”，不记录明文凭据。

## 安装与配置要求

必须准备以下配置：

- `api_username`
- `api_password`
- `sdk_download_url`
- `sdk_local_path`
- `http_api_manual_path`
- `install_root`
- `api_endpoint`

推荐运行时约定：

- `THS_API_USERNAME`
- `THS_API_PASSWORD`
- `THS_SDK_PATH`
- `THS_HTTP_MANUAL_PATH`

## 用户提供信息

- 官方下载地址：
  [同花顺下载中心](https://quantapi.10jqka.com.cn/gwstatic/static/ds_web/quantapi-web/download.html)
- 账号密码：
  用户已提供，但不得写入仓库，请仅作为运行时配置使用

## 强制执行步骤

### Step 1：安装 SDK

1. 使用用户本地 SDK 包，或从官方下载中心获取 Linux SDK
2. 在目标目录完成解压或安装
3. 记录安装路径
4. 验证 SDK 文件存在

### Step 2：配置认证

1. 从运行时配置读取账号密码
2. 完成登录配置
3. 验证登录成功

### Step 3：接口可用性测试

1. 调用一次最小可用接口验证服务联通
2. 记录测试结果

### Step 4：提取价格数据

提取：

- 标的：平安银行
- 代码：`000001`
- 时间范围：自 `2026-01-01` 至执行当日
- 字段：至少包含交易日期、收盘价

保存到：

- `outputs/data/F11_000001_close_prices.csv`

### Step 5：提取公告数据

提取：

- 标的：中国神华
- 代码：`601088`
- 时间范围：自 `2026-02-13` 至执行当日
- 字段：
  - 公告日期
  - 公告标题
  - 公告链接

保存到：

- `outputs/data/F11_601088_announcements.json`

### Step 6：生成报告

报告必须包含：

1. SDK 安装路径
2. 认证是否成功
3. 接口连通性结果
4. 价格数据提取结果摘要
5. 公告数据提取结果摘要
6. 异常与风险说明

保存到：

- `outputs/data/F11_installation_report.md`

## 通过标准

1. SDK 已真实安装
2. 登录配置成功
3. 价格数据 CSV 文件存在且来自真实 API
4. 公告数据 JSON 文件存在且来自真实 API
5. 报告存在且引用真实提取结果

## 失败标准

1. 未安装 SDK
2. 未完成登录配置
3. 只生成 Markdown 说明，未产出真实数据文件
4. 数据文件为手工拼接内容而非 API 结果
5. 在仓库文件中写入账号或密码
