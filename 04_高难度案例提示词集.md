## F11 细化测试目标

验证 OpenClaw 能否使用真实同花顺 Linux SDK 完成安装、登录和数据提取，而不是仅根据案例描述生成一份看起来合理的说明文档。

## F11 强制要求

1. 必须真实安装同花顺 SDK。
2. 必须使用运行时配置中的账号密码完成登录。
3. 不允许把账号密码写入仓库文件。
4. 必须提取两类真实数据：
   - 平安银行自 2026-01-01 以来的收盘价
   - 中国神华自 2026-02-13 以来的公告日期、标题、链接
5. 必须生成三份产物：
   - `outputs/data/F11_installation_report.md`
   - `outputs/data/F11_000001_close_prices.csv`
   - `outputs/data/F11_601088_announcements.json`
6. 如果只有 Markdown 报告，没有真实数据文件，直接判失败。

## F11 细化提示词

```text
【测试案例 F11】同花顺 API 接口配置与公告数据提取

你必须使用真实同花顺 SDK 完成安装、认证、连通性测试和数据提取，不允许只写一份 Markdown 说明文档来冒充执行结果。

【用户提供条件】
- 已提供同花顺账号和密码，但这些凭据只能作为运行时配置使用，禁止写入仓库文件
- 官方 SDK 下载地址：
  https://quantapi.10jqka.com.cn/gwstatic/static/ds_web/quantapi-web/download.html
- 用户也可能已提供本地 Linux SDK 和 HTTP API 用户手册

【强制执行步骤】
Step 1 - SDK 准备
1. 优先检查是否已有本地 Linux SDK 包
2. 若无本地包，则使用官方地址下载 SDK
3. 在指定目录完成安装
4. 验证 SDK 文件存在

Step 2 - 认证配置
1. 从运行时配置读取账号密码
2. 完成登录配置
3. 测试登录是否成功
4. 不要在任何仓库文件中写入账号或密码

Step 3 - 接口连通性测试
1. 调用最小可用接口验证服务联通
2. 记录连通性结果

Step 4 - 提取价格数据
提取平安银行 000001 自 2026-01-01 以来的收盘价数据
输出文件：
outputs/data/F11_000001_close_prices.csv

Step 5 - 提取公告数据
使用同花顺问财或公告相关接口，提取中国神华 601088 自 2026-02-13 以来的：
- 公告日期
- 公告标题
- 公告链接

输出文件：
outputs/data/F11_601088_announcements.json

Step 6 - 生成安装与测试报告
报告必须包含：
- SDK 安装路径
- 认证状态
- 接口连通性结果
- 价格数据提取摘要
- 公告数据提取摘要
- 异常和风险说明

输出文件：
outputs/data/F11_installation_report.md

【通过标准】
- SDK 已真实安装
- 登录成功
- 价格 CSV 存在
- 公告 JSON 存在
- 报告存在且引用真实提取结果

【失败标准】
- 未安装 SDK
- 未完成登录
- 只有 Markdown 报告，没有真实数据文件
- 把账号密码写进仓库文件
- 数据文件不是 API 实际结果
```
